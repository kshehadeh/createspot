On Create Spot you can explore creator portfolios, community exhibits, and the main feed.

![Create Spot homepage](/images/homepage.png)

## First steps

1. **Sign in** (optional for browsing; required for some actions).
2. **Browse exhibits** — See [Exhibits](/browsers/exhibits) for permanent and temporary exhibits and how to use Grid and Path views.

![Exhibition list](/images/exhibition-list.png)

3. **Discover creators** — Visit creator profiles and portfolios from exhibits and the feed.
4. **Explore the gallery** — Open the [permanent exhibit grid](/inspire/exhibition/gallery/grid) to browse public portfolio work.

![Permanent exhibit — grid view](/images/exhibition-grid.png)
