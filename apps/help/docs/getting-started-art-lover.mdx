On Create Spot you can explore work from creators inspired by weekly prompts and browse exhibits.

## First steps

1. **Sign in** (optional for browsing; required for some actions).
2. **Browse exhibits** — See [Exhibits](/browsers/exhibits) for permanent and temporary exhibits and how to use Grid and Map views.
3. **Discover creators** — Visit creator profiles and portfolios from exhibits and prompt galleries.
4. **Explore prompts** — See [Inspiration > Prompts](/inspiration/prompts) to view submissions for each week's prompt.
